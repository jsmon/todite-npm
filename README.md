# `todite` v0.1
`todite` is an npm package created to let you intergrate your JavaScript / TypeScript projects with [Todite](https://todite.now.sh/), a free to-do list app.

**Warning: This is not a stable version**

The full documentation for the package is available at [todite.now.sh/docs/npm/v1.0](https://todite.now.sh/docs/npm/v1.0), and the documentation for the CLI is available at [todite.now.sh/docs/cli/v1.0](https://todite.now.sh/docs/cli/v1.0)
